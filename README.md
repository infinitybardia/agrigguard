# Agrigguard

Static deployment of the AgriGuard AI Pro dashboard.

## Deployment

This project contains a single static `index.html` file and can be hosted on GitHub Pages or any static site host.

### GitHub Pages

1. Push the `main` branch.
2. In repository settings, enable GitHub Pages from the `main` branch.

### Local Preview

Open `index.html` in a browser or run a local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.
